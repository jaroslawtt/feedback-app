## Running the app

### Client

---
$ cd client

$ npm run start | yarn run start

---

### Server

---

$ cd server

$ docker compose up

Add .env file with db url configuration:
DATABASE_URL="postgresql://postgres:123@localhost:5432/mydb?schema=public"

$ npm run start | yarn run start
